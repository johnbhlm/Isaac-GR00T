ControllerMessageParams

class humanoid_sdk_py.ControllerMessageParams

Bases: pybind11_object

Controller status structure

Note

  • controller_enabled_state: Controller enabled status; 0: Controller disabled; 1: Controller enabled;

  • power_state: Robot power information; 0: Powered on; 1: Powered off;

  • brake_state: Brake information for each joint/component; true: Brake engaged (holding), false: Brake released. Each bit represents a different joint or component, Bits 0-17 are valid, bits 18-31 are reserved.

  • reserved: Reserved field
