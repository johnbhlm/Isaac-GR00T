humanoid::h10w::RealtimeJointsParams

struct RealtimeJointsParams

Realtime joint motion control parameters structure.

  • left_arm: 7 joint angles of left arm (radians), valid when left_arm_valid=1

  • left_arm_valid: Left arm data validity flag: 1-execute motion, 0-maintain current position

  • right_arm: 7 joint angles of right arm (radians), valid when right_arm_valid=1

  • right_arm_valid: Right arm data validity flag: 1-execute motion, 0-maintain current position

  • torso: 3 joint angles of torso (radians), valid when torso_valid=1

  • torso_valid: Torso data validity flag: 1-execute motion, 0-maintain current position

  • time: Motion control period (seconds)

humanoid::h10w::RealtimeTcpParams

struct RealtimeTcpParams

Realtime TCP motion control parameters structure.

  • left_arm: Left arm TCP pose: [x, y, z, roll, pitch, yaw] in meters and radians

  • left_arm_valid: Left arm data validity: 1-execute motion, 0-maintain current pose (meters and radians)

  • right_arm: Right arm TCP pose: [x, y, z, roll, pitch, yaw] in meters and radians

  • right_arm_valid: Right arm data validity: 1-execute motion, 0-maintain current pose (meters and radians)

  • torso: Torso TCP pose: [x, y, z, roll, pitch, yaw] in meters and radians

  • torso_valid: Torso data validity: 1-execute motion, 0-maintain current pose (meters and radians)

  • time: Motion control period (seconds)
