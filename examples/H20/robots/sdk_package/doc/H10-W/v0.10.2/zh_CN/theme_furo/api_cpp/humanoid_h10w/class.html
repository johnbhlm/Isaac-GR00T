int setControlPolicy(const ControlPolicy &policy)¶

设置机器人控制策略

控制策略只会影响笛卡尔空间运动，关节运动不受影响

  • FULL_BODY: 全身控制策略，躯干可能随左右臂笛卡尔运动而运动

  • SINGLE_ARM: 单臂控制策略，只能进行左右臂的笛卡尔运动

返回:

int

  • 0: 成功；

  • 其他: 错误码；

参数:

policy – [in] 控制策略，FULL_BODY 或 SINGLE_ARM

int setSafeMode(bool mode)¶

启用或禁用机器人安全模式

在安全模式下，可以将超过限位的关节恢复到正常范围，以确保操作安全。

备注

安全模式需要禁用实时指令才能生效

参数:

mode – [in] 安全模式状态，true 启用安全模式，false 禁用安全模式

返回:

int

  • 0: 成功；

  • 其他: 错误码；

int getSafeMode(bool &mode)¶

获取机器人安全模式状态

参数:

mode – [out] 安全模式状态

返回:

int

  • 0: 成功；

  • 其他: 错误码；

备注

控制器模式说明

  • EnableController = true: 启用运动控制功能，可以使用高级运动控制API

  • EnableController = false: 激活底层控制模式，直接控制电机底层

在使用运动控制功能前，请确保已正确设置控制器模式。
