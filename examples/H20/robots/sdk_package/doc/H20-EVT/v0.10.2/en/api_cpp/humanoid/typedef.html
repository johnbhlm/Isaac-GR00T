humanoid::JointTarget

using humanoid::JointTarget = std::tuple<uint32_t, double>

Represents the joint motion as an typedef.

  • tuple_element 0: joint_index

  • tuple_element 1: target_position

humanoid::JointLimit

using humanoid::JointLimit = std::vector<std::tuple<uint32_t, double, double>>

Represents the joint limit parameters as an typedef.

  • tuple_element 0: joint_index（range:1~19)

  • tuple_element 1: max joint limit value

  • tuple_element 2: min joint limit value
