  • h10w.R_ARM_JOINT1 = 8

  • h10w.R_ARM_JOINT6 = 13

  • h10w.L_GRIPPER_MOTOR = 18

  • h10w.R_GRIPPER_MOTOR = 19

  • h10w.SIZE = 20

CartIndex

class humanoid_sdk_py.h10w.CartIndex

笛卡尔坐标系下工具中心点的枚举类型

字段说明:

  • UNKNOWN_CART: 未知工具中心点

  • LEFT_ARM: 左臂工具中心点

  • RIGHT_ARM: 右臂工具中心点

  • TORSO: 躯干工具中心点
