Python Usage Examples

This document describes how to use Python to call the HumanoidSDK library and demonstrates building a simple example program.

Hint

For more examples, see the sdk_package/example_py directory.

Configure environment variables

cd ~/sdk_package
export LD_LIBRARY_PATH="$(pwd)/lib/linux/x86_64/x64:$(pwd)/dependency/sdk/Runtime_Env/linux/x86_64/x64:$(pwd)/dependency/fastdds/linux/x86_64/x64:$LD_LIBRARY_PATH"
export PYTHONPATH="$(pwd)/lib/linux/x86_64/x64:$PYTHONPATH"

Optional: To use a custom DDS configuration file, set the ROBOT_DDS_PROFILE environment variable:

export ROBOT_DDS_PROFILE=path/to/your/Robot_DDS_Profile.xml

Build a simple example program

1. Create the example.py file

 1 ## example.py
 2 import humanoid_sdk_py
 3 import humanoid_sdk_py.h10w as h10w
 4 import time
 5
 6 h10w_params = h10w.H10wParams()
 7 time.sleep(1)
 8
 9 ret, limit = h10w_params.getJointSoftLimit()
10 if ret == 0:
11     for index, max, min in limit:
12         print(
13             "    joint {}: max pos= {:.3f} rad, min pos= {:.3f} rad".format(
14                 index, max, min
15             )
16         )
17 else:
18     print("Failed to get joint soft limit.")

2. Run the example program

python3 example.py
