Introduction

Overview

The H23-EVT SDK internally uses FAST-DDS as middleware to communicate with the robot controller, enabling functional calls for user convenience. It supports both C++ and Python programming languages, multiple robot models, and simultaneous connections to multiple robots. For different robot models, the SDK categorizes functional modules into broad classes such as parameters, motion, and system, allowing users to quickly locate the desired interfaces.
