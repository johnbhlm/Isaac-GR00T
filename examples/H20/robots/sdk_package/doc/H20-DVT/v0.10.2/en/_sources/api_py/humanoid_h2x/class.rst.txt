.. admonition:: 参数说明

    - **mode**： 电机控制模式，0 = Disable（命令忽略），1 = Enable（命令生效）。

    - **kp**：关节刚度系数

    - **kd**：关节阻尼系数

    - **reserved**：预留

IMUState
----------------------

.. autoclass:: humanoid_sdk_py.h2x.IMUState
   :members:
   :undoc-members:
   :show-inheritance:
   :member-order: bysource

.. admonition:: 参数说明
  :class: note

  - **quaternion**： 四元数，表示机器人坐标系绕世界坐标系的转动情况。

    - **0**： w
    - **1**： x
    - **2**： y
    - **3**： z
  - **gyroscope**： 陀螺仪角速度（绕三个坐标轴），单位 **rad/s**。

    - **0**： x轴
    - **1**： y轴
    - **2**： z轴
  - **accelerometer**： 加速度（三个坐标轴），单位 **m/s²**。

    - **0**： x轴
    - **1**： y轴
    - **2**： z轴
  - **rpy**： 欧拉角 (Roll, Pitch, Yaw)，表示机器人坐标系绕世界坐标系的转动角度，单位 **rad**。

    - **0**： roll
    - **1**： pitch
    - **2**： yaw

  - **temperature**： IMU传感器温度，单位 **°C**。
  - **error_code**： IMU错误码。
  - **warning_code**： IMU警告码。
