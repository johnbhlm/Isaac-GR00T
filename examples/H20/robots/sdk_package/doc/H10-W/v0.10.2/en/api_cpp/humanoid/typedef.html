Typedef

humanoid::JointTarget

using humanoid::JointTarget = std::tuple<uint32_t, double>

Represents the joint motion as an typedef.

  • tuple_element 0: joint_index

  • tuple_element 1: target_position

humanoid::JointLimit

using humanoid::JointLimit = std::vector<std::tuple<uint32_t, double, double>>

Represents the joint limit parameters as an typedef.

  • tuple_element 0: joint_index（range:1~19)

  • tuple_element 1: max joint limit value

  • tuple_element 2: min joint limit value

humanoid::JointMaxAcc

using humanoid::JointMaxAcc = std::vector<std::tuple<uint32_t, double>>

Represents the joint maximum acceleration as an typedef.

  • tuple_element 0: joint_index（range:1~19)

  • tuple_element 1: maximum joint acceleration

humanoid::JointMaxVel

using humanoid::JointMaxVel = std::vector<std::tuple<uint32_t, double>>

Represents the joint maximum velocity as an typedef.

  • tuple_element 0: joint_index（range:1~19)

  • tuple_element 1: maximum joint velocity

humanoid::CartTransMaxAcc

using humanoid::CartTransMaxAcc = std::vector<std::tuple<uint32_t, double>>

Represents the cartesian translation maximum acceleration as an typedef.

  • tuple_element 1: maximum cartesian translation acceleration
