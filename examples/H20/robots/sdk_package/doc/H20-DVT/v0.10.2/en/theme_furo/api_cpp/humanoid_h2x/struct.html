humanoid::h2x::MotorCmd¶

struct MotorCmd¶

Full-body motor control (arm + head motor) parameters structure.

Public Members

uint8_t mode¶
uint8_t operation_mode¶
float q¶
float dq¶
float tau¶
float kp¶
float kd¶
std::array<uint32_t, 4> reserved¶

Parameter Description

  • mode_: Motor control mode, 0 = Disable (command ignored), 1 = Enable (command active).

  • operation_mode_: Operation mode.

Operation Mode (Hex)

Mode Description

0x00

Force-Position Hybrid Control Mode (Default Operation Mode)

0x01

Synchronous Position Control Mode

0x02

Synchronous Velocity Control Mode

0x03

Synchronous Motor Torque Control Mode

  • q: Joint target position, unit: rad

  • dq: Joint target velocity, unit: rad/s

  • tau: Joint feedforward torque, unit: Nm

  • kp: Joint stiffness coefficient

  • kd: Joint damping coefficient

  • reserved: Reserved

humanoid::h2x::IMUState¶

struct IMUState¶

IMU state parameters structure.

Public Members

std::array<float, 4> quaternion¶
std::array<float, 3> gyroscope¶
std::array<float, 3> accelerometer¶
std::array<float, 3> rpy¶
float temperature¶
int32_t error_code¶
int32_t warning_code¶

Parameter Description

  • quaternion_[4]: Quaternion, representing the rotation of the robot coordinate system relative to the world coordinate system.

    • 0： w

    • 1： x

    • 2： y

    • 3： z

  • gyroscope_[3]: Gyroscope angular velocity (around the three coordinate axes), unit rad/s.

    • 0: x-axis

    • 1: y-axis

    • 2: z-axis

  • accelerometer_[3]: Acceleration (along the three coordinate axes), unit m/s².

    • 0: x-axis

    • 1: y-axis

    • 2: z-axis

  • rpy_[3]: Euler angles (Roll, Pitch, Yaw), representing the rotation angles of the robot coordinate system relative to the world coordinate system, unit rad.

    • 0： roll

    • 1： pitch

    • 2： yaw

  • temperature_: IMU sensor temperature, unit °C.

  • error_code_: IMU error code.

  • warning_code_: IMU warning code.

Parameter Description

  • mode_: Current control mode, 0 = Disable (command ignored), 1 = Enable (command active).

  • operation_mode_: Operation mode.

Operation Mode (Hex)

Mode Description

0x00

Force-Position Hybrid Control Mode (Default Operation Mode)

0x01

Synchronous Position Control Mode

0x02

Synchronous Velocity Control Mode

0x03

Synchronous Motor Torque Control Mode

  • q_: Joint feedback position, unit rad.

  • dq_: Joint feedback velocity, unit rad/s.

  • ddq_: Joint feedback acceleration, unit rad/s².

  • cur_: Joint feedback current, unit A.

  • tau_est_: Joint feedback torque, unit N·m.

  • temperature_[2]: Joint temperature, unit °C.

    • 0: Surface temperature

    • 1: Winding temperature

  • kp_: Joint stiffness coefficient.

  • kd_: Joint damping coefficient.

  • error_code_: Error code.

  • warning_code_: Warning code.

  • reserved_[4]: Reserved, 4 entries of unsigned long type.

Parameter Description

  • version: Software version information

  • mode_pr: Parallel mechanism (ankle and waist) control mode

    • 0: PR mode (Pitch/Roll angle control) [default]

    • 1: AB mode (direct motor control)

  • tick: Timer, increments every 1ms

  • imu_state: IMU state

  • motor_state: All motor states

  • has_battery_info: Whether battery information is present

    • true: Contains battery information

    • false: Does not contain battery information

  • battery_info: Battery information

  • ground_count: Number of foot-ground contact switches

  • ground_signal: Foot-ground contact switch status

  • time_seconds: Cumulative time, unit s

  • is_pcu_valid: Whether the PCU is valid

    • 0: Invalid

    • 1: Valid

  • pcu_state: PCU state

  • is_cdu_valid: Whether the CDU is valid

    • 0: Invalid

    • 1: Valid

  • cdu_state: CDU state

  • is_data_valid: Whether the data is valid

    • 0: All invalid

    • 1: Partially valid

    • 2: All valid

  • error_code: Driver layer error code

    • 0: No error

    • -2000~-2999: Driver layer software error

    • -1: Firmware layer error
