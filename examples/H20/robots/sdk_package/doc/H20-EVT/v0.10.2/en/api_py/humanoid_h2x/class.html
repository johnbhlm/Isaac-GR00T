Operation Mode (Hex)

Mode Description

Force-Position Hybrid Control Mode (Default Operation Mode)

Parameter Description

  • quaternion: Quaternion, representing the rotation of the robot coordinate system relative to the world coordinate system.

    • 0： w

    • 1： x

    • 2： y

    • 3： z

  • gyroscope: Gyroscope angular velocity (around the three coordinate axes), unit rad/s.

    • 0: x-axis

    • 1: y-axis

    • 2: z-axis

  • accelerometer: Acceleration (three coordinate axes), unit m/s².

    • 0: x-axis

    • 1: y-axis

    • 2: z-axis

  • rpy: Euler angles (Roll, Pitch, Yaw), representing the rotation angles of the robot coordinate system relative to the world coordinate system, unit rad.

    • 0： roll

    • 1： pitch

    • 2： yaw

  • temperature: IMU sensor temperature, unit °C.

  • error_code: IMU error code.

  • warning_code: IMU warning code.

Rise(self: humanoid_sdk_py.h2x.H2xHighLevel) → int

robot rise up interface

The robot moves from a lying posture to a standing posture

Execution conditions: The ground is flat and there is enough space, and the posture is manually unfolded;

Execution process:

  1. The initial state is damping mode, and the posture is adjusted to a standard lying posture through position control.

  2. Slowly rise from lying to standing posture, position control is not anti-interference.

  3. Squat down smoothly and maintain anti-interference state.

Execution completed: The robot stands up smoothly and maintains balance mode;

Returns:

code: 0 indicates success, other indicates failure

Return type:

int

SitDown(self: humanoid_sdk_py.h2x.H2xHighLevel) → int

robot sit down interface

The robot moves from a standing posture to a sitting posture.

Execution conditions: The robot is in a standing state

Execution process: Position control performs the sit-down action;

Execution completed: The upper body joint is in damping mode, the shoulder joint and lower body are in locked position mode, maintaining a stable sitting posture;

Returns:

code: 0 indicates success, other indicates failure

SquatDown(self: humanoid_sdk_py.h2x.H2xHighLevel) → int

robot squat down interface

The robot smoothly squats down from a standing posture to a squatting posture.

Execution conditions: The robot is in a standing state, the ground is flat and there are no obstacles.

Execution process:

  1. Initially in walk-run mode;

  2. First switch to the squat model and perform the squat action;

  3. Maintain anti-interference state during movement;

Execution completed: Finally, the feet maintain stiffness, and the other joints.

Returns:

code: 0 indicates success, other indicates failure

Return type:

int

robot squat up interface

The robot smoothly stands up from a squatting posture to a standing posture.

Execution process:

  2. Then switch to the stand-up model;

  3. During movement, maintain anti-interference state;
