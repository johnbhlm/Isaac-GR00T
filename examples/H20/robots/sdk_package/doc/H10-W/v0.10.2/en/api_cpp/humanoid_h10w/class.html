int setControlPolicy(const ControlPolicy &policy)

Set the robot control policy.

control policy only affects Cartesian space motion of the body, while joint motion is not affected.

  • FULL_BODY: full-body control policy, the body may move in Cartesian space while moving the arms.

  • SINGLE_ARM: single-arm control policy, only the left and right arms

Returns:

int

  • 0: Success;

  • Other: Error code;

Parameters:

policy – [in] Control policy to set (FULL_BODY or SINGLE_ARM)

int getControlPolicy(ControlPolicy &policy)

Get the robot current control policy.

control policy only affects Cartesian space motion of the body, while joint motion is not affected.

  • FULL_BODY: full-body control policy, the body may move in Cartesian space while moving the arms.

  • SINGLE_ARM: single-arm control policy, only the left and right arms

Returns:

int

  • 0: Success;

  • Other: Error code;

Parameters:

policy – [out] Reference to store the current control policy

int setSafeMode(bool mode)

Enable or disable robot safe mode.

In safe mode, you can drive any joint beyond its limits back towards the normal range to ensure operational safety.

Note

safe mode requires disabling realtime cmd to take effect

Parameters:

mode – [in] current safe mode status, true to enable safe mode, false to disable safe mode

Returns:

int

  • 0: Success;

  • Other: Error code;

int getSafeMode(bool &mode)

Parameters:

mode – [out] Reference to store the current safe mode status

Returns:

int

  • 0: Success;

  • Other: Error code;

Note

Controller mode description

  • EnableController = true: Enable motion control features and use high-level motion control APIs

  • EnableController = false: Activate low-level control mode and directly control the motor layer

Before using motion control features, make sure the controller mode is set correctly.
