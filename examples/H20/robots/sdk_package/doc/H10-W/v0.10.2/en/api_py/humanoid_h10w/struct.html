Struct

RealtimeJointsParams

class humanoid_sdk_py.h10w.RealtimeJointsParams

Realtime joint motion control parameters structure

Field description:

  • left_arm: 7 joint angles (rad) of the left arm

  • left_arm_valid: Validity of left arm data, 1-execute, 0-hold

  • right_arm: 7 joint angles (rad) of the right arm

  • right_arm_valid: Validity of right arm data

  • torso: 3 joint angles (rad) of the torso

  • torso_valid: Validity of torso data

  • time: Motion control cycle (seconds)

RealtimeTcpParams

class humanoid_sdk_py.h10w.RealtimeTcpParams

Realtime TCP motion control parameters structure

Field description:

  • left_arm: TCP pose of left arm (x, y, z, rx, ry, rz) (m, rad)

  • left_arm_valid: Validity of left arm data, 1-execute, 0-hold

  • right_arm: TCP pose of right arm (x, y, z, rx, ry, rz) (m, rad)

  • right_arm_valid: Validity of right arm data

  • torso: TCP pose of torso (x, y, z, rx, ry, rz) (m, rad)

  • torso_valid: Validity of torso data

  • time: Motion control cycle (seconds)

LinearTarget

class humanoid_sdk_py.h10w.LinearTarget

TCP linear motion control parameters structure

Field description:

  • type: CartIndex enumeration type, 1-left arm 2-right arm 3-torso

  • pose: 6-dimensional array [x, y, z, rx, ry, rz] (position and rotation)

  • velocityPercent: Velocity percentage (0-1]

  • accelerationPercent: Acceleration percentage (0-1]

TcpPoseParams

class humanoid_sdk_py.h10w.TcpPoseParams

TCP pose parameters structure

Field description:

  • type: CartIndex enumeration type, 1-left arm 2-right arm 3-torso

  • pose: 6-dimensional array [x, y, z, rx, ry, rz] (position and rotation)

MoveMessageParams

class humanoid_sdk_py.h10w.MoveMessageParams

Motion state parameters structure

Field description:

  • state: Motion state, 0-Idle 1-Running 2-Paused 3-Error

  • position: Position vector

  • velocity: Velocity vector

  • tcp_pose: TCP pose parameters vector

  • task_id: Instruction sequence number, identifies the current instruction

ErrorMessageParams

class humanoid_sdk_py.h10w.ErrorMessageParams

Error message parameters structure

Field description:

  • id: Error ID

  • level: Error level

  • module: Error module level, 1-Controller 2-Driver 3-Firmware

  • error_code: Error code

  • msg: Error message
