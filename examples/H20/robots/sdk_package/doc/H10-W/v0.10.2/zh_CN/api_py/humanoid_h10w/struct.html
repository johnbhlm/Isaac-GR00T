结构体

RealtimeJointsParams

class humanoid_sdk_py.h10w.RealtimeJointsParams

实时指令关节运动控制参数结构体

字段说明:

  • left_arm: 左臂 7 个关节角（rad）

  • left_arm_valid: 左臂数据有效性，1-执行，0-保持

  • right_arm: 右臂 7 个关节角（rad）

  • right_arm_valid: 右臂数据有效性

  • torso: 躯干 3 个关节角（rad）

  • torso_valid: 躯干数据有效性

  • time: 运动控制周期（秒）

RealtimeTcpParams

class humanoid_sdk_py.h10w.RealtimeTcpParams

实时指令末端位姿运动控制参数结构体

字段说明:

  • left_arm: 左臂末端位姿 (x, y, z, rx, ry, rz) (m, rad)

  • left_arm_valid: 左臂数据有效性，1-执行，0-保持

  • right_arm: 右臂末端位姿 (x, y, z, rx, ry, rz) (m, rad)

  • right_arm_valid: 右臂数据有效性

  • torso: 躯干末端位姿 (x, y, z, rx, ry, rz) (m, rad)

  • torso_valid: 躯干数据有效性

  • time: 运动控制周期（秒）

LinearTarget

class humanoid_sdk_py.h10w.LinearTarget

末端直线运动控制参数结构体

字段说明:

  • type: CartIndex枚举类型，1-左臂 2-右臂 3-躯干

  • pose: 6维数组 [x, y, z, rx, ry, rz]（位置和旋转）

  • velocityPercent: 速度百分比 (0-1]

  • accelerationPercent: 加速度百分比 (0-1]

TcpPoseParams

class humanoid_sdk_py.h10w.TcpPoseParams

末端位姿参数结构体

字段说明:

  • type: CartIndex枚举类型，1-左臂 2-右臂 3-躯干

  • pose: 6维数组 [x, y, z, rx, ry, rz]（位置和旋转）

MoveMessageParams

class humanoid_sdk_py.h10w.MoveMessageParams

运动状态参数结构体

字段说明:

  • state: 运动状态，0-Idle 1-Running 2-Paused 3-Error

  • position: 位置向量

  • velocity: 速度向量

  • tcp_pose: TCP位姿参数向量

  • task_id: 指令序列号，标识当前正在执行的指令序号

ErrorMessageParams

class humanoid_sdk_py.h10w.ErrorMessageParams

错误信息参数结构体

字段说明:

  • id: 错误ID

  • level: 错误级别

  • module: 错误模块级别，1-控制器模块 2-驱动模块 3-固件模块

  • error_code: 错误代码

  • msg: 错误消息
