class humanoid_sdk_py.h2x.MotorCmd

基类：pybind11_object

电机控制命令结构体

包含电机控制模式、位置、速度、力矩等参数。

property dq
property kd
property kp
property mode
property operation_mode
property q
property reserved
property tau

参数说明

  • mode： 电机控制模式，0 = Disable（命令忽略），1 = Enable（命令生效）。

  • operation_mode： 工作模式。

工作模式 (Hex)

模式描述

0x00

力位混控模式（默认工作模式）

0x01

伺服位置控制模式

0x02

伺服速度控制模式

0x03

电流、力矩控制模式和刹车控制指令

0x04

配置指令

0x05

电机控制参数查询指令

  • q：关节目标位置，单位：rad

  • dq：关节目标速度，单位：rad/s

  • tau：关节前馈力矩，单位：Nm

  • kp：关节刚度系数

  • kd：关节阻尼系数

  • reserved：预留

IMUState

class humanoid_sdk_py.h2x.IMUState

基类：pybind11_object

IMU状态结构体

包含IMU传感器数据，如四元数、角速度、加速度等。

property accelerometer
property error_code
property gyroscope
property quaternion
property rpy
property temperature
property warning_code

参数说明

  • quaternion： 四元数，表示机器人坐标系绕世界坐标系的转动情况。

    • 0： w

    • 1： x

    • 2： y

    • 3： z

  • gyroscope： 陀螺仪角速度（绕三个坐标轴），单位 rad/s。

    • 0： x轴

    • 1： y轴

    • 2： z轴

  • accelerometer： 加速度（三个坐标轴），单位 m/s²。

    • 0： x轴

    • 1： y轴

    • 2： z轴

  • rpy： 欧拉角 (Roll, Pitch, Yaw)，表示机器人坐标系绕世界坐标系的转动角度，单位 rad。

    • 0： roll

    • 1： pitch

    • 2： yaw

  • temperature： IMU传感器温度，单位 °C。

  • error_code： IMU错误码。

  • warning_code： IMU警告码。

    • 1：有效

  • is_cdu_valid：CDU 是否有效(此机型无效)

    • 0：无效

    • 1：有效

  • cdu_state：CDU 状态

  • is_data_valid：数据是否有效

    • 0：全部无效

    • 1：部分有效

    • 2：全部有效

  • error_code：驱动层错误码

    • 0：无错误

    • -2000~-2999：驱动层软件错误

错误码

说明
