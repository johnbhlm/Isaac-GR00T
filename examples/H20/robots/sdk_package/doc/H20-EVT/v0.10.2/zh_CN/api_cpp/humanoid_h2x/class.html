humanoid::h2x::H2xLowLevel

class H2xLowLevel

H2x 机器人低级别控制接口类

该类提供了对 H2x 机器人底层硬件的直接控制接口，包括关节空间控制、 读取机器人状态等功能。适用于需要精细控制机器人底层行为的应用场景。

Public Functions

int writeLowCmd(const LowCmd &lowCmd)

写入低级别控制命令

向机器人发送低级别控制命令，包括关节角度、有效性标志和时间戳。 该接口提供对机器人底层硬件的直接控制。

参数:

lowCmd – [in] 低级别控制命令结构体

返回:

int 执行结果

  • 0: 成功；

  • 其他: 错误码；

int 执行结果

从LR-CU（左右控制单元）读取当前状态信息。

参数:

返回:

int 执行结果

  • 0: 成功；

  • 其他: 错误码；

备注

控制器模式说明

  • EnableController = true: 启用运动控制功能，可以使用高级运动控制API

  • EnableController = false: 激活底层控制模式，直接控制电机底层

在使用运动控制功能前，请确保已正确设置控制器模式。

int32_t Move(double vx, double vy, double vyaw, double height, bool continuous_move)

int32_t SquatDown()

机器人从站立姿态平稳下蹲到蹲姿态接口 执行条件：机器人处于站立状态，地面平坦且无障碍物； 执行过程： 1.初始为走跑模式； 2.先切换至下蹲模型，执行下蹲动作； 3.运动过程中保持抗干扰状态； 执行完成：最终脚部保持刚度，其余关节切换为阻尼模式；

返回:

int32_t

  • 0: 成功；

  • 其他: 错误码；

int32_t SquatUp()

机器人从蹲姿平稳站起到站立姿态接口 执行条件：机器人处于稳定蹲姿，周围无障碍物； 执行过程： 1.先切换为位置模式，移动至蹲姿初始位置； 2.后切换为蹲起模型； 3.移动过程中保持抗干扰状态； 执行完成：机器人处于稳定站姿，并自动切换为走跑模式；

返回:

int32_t

  • 0: 成功；

  • 其他: 错误码；

int32_t SitDown()

机器人从站立姿态移动到坐姿态接口 执行条件：机器人处于站立状态 执行过程：位置控制执行坐下动作； 执行完成：机器人上半身关节为阻尼模式，腰部关节、下半身锁定位置模式，保持稳定坐姿；

返回:

int32_t

  • 0: 成功；

  • 其他: 错误码；

int32_t Rise()

机器人从躺姿移动到站立姿态接口 执行条件：地面平坦且有足够空间，人为手动将姿势展开； 执行过程： 1.初始状态为阻尼模式，先通过位置控制将姿态调整至标准躺姿。 2.从躺姿缓慢至蹲姿，位置控制非抗干扰； 3.从蹲姿平稳站立，保持抗干扰状态； 执行完成：机器人站立完成后保持平衡模式；

返回:

int32_t

  • 0: 成功；

  • 其他: 错误码；

int32_t realtimeMove(const RealtimeCmd &realtimeCmd)

发送实时控制命令

向机器人发送实时控制命令，使用实时DDS消息进行通信。 当前该接口主要用于 ClimbStairsMode 下的上半身实时控制。 realtimeCmd.time 表示本条目标的插补时长，单位为秒；例如 0.1 表示控制器会在 100 ms 内平滑插补到目标。

参数:

realtimeCmd – [in] 实时控制命令结构体

返回:

int32_t

  • 0: 成功；

  • 其他: 错误码；

备注

控制器模式说明

  • EnableController = true: 启用运动控制功能，可以使用高级运动控制API

  • EnableController = false: 激活底层控制模式，直接控制电机底层

在使用运动控制功能前，请确保已正确设置控制器模式。
